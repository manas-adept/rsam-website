# RSAM Website – Roller Sports Association Moradabad

Static website for RSAM, hosted on GitHub Pages.

## Files

```
rsam-website/
├── index.html        ← Main page (all sections)
├── styles.css        ← All styles
├── script.js         ← Animations & interactions
├── images/           ← Drop your photos here
│   ├── president.jpg
│   ├── secretary.jpg
│   ├── treasurer.jpg
│   ├── vp.jpg
│   ├── joint_sec.jpg
│   ├── technical.jpg
│   ├── member1.jpg … member4.jpg
│   ├── certificate.jpg   ← Registration certificate scan
│   ├── highlight1.jpg … highlight5.jpg
│   └── news_main.jpg
└── README.md
```

## How to deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `rsam-website`)
2. Push all files in this folder to the `main` branch
3. Go to **Settings → Pages → Source → Deploy from branch → main / root**
4. Your site will be live at `https://<your-username>.github.io/rsam-website/`

## Customisation checklist

- [ ] Replace all `[Placeholder Name]` values in `index.html` with real names
- [ ] Drop photos into `images/` folder matching the filenames above
- [ ] Replace certificate placeholder with `<img src="images/certificate.jpg" class="cert-img" />`
- [ ] Update YouTube URL: `href="https://www.youtube.com/@YourChannelHandle"`
- [ ] Update Instagram URL: `href="https://www.instagram.com/YourHandle"`
- [ ] Fill in contact email, phone, and address
- [ ] Fill in Registration No. and validity date in the Certificate section
- [ ] Add real news/circular entries in the News section
- [ ] Add real highlight images and captions
